# No-Overlap Directional Vehicle Icons

This version fixes the overlapping issue.

Rules:
- No text inside the train/metro/tram pictogram.
- No rails near the destination label.
- Destination label sits inside a dedicated bottom pill.
- `←` is used for coming/returning to the left-side terminal.
- `→` is used for going toward the right-side terminal.

Examples:
- P1A ← A. Liosia
- P1A → Airport
- M1 ← Piraeus
- M1 → Kifissia
- T7 ← Akti Pos.
- T7 → Voula

# Athens Transit Icons And Rules

## Included Assets

- Updated station and network reference: `athens_fixed_rail_station_coordinates.md`
- Full icon pack: `athens_transit_t7_station_icons_updated/`

## Station Icon Rules

- Use station icons from `athens_transit_t7_station_icons_updated/station_smart_codes/athens_station_smart_code_icons/`.
- Apply these icons to all fixed-rail stations across metro, tram and suburban railway.
- For shared stations with multiple lines or modes, use the icon of the larger, more important line.
- In practice, metro takes priority over tram or suburban at major interchanges such as Syntagma, Monastiraki, Piraeus, Airport and Doukissis Plakentias.

## Vehicle Icon Rules

- Use directional vehicle icons from `athens_transit_t7_station_icons_updated/directional_vehicle_icons/directional/` whenever the route and direction are known.
- If a suburban train destination or direction is unknown, use `athens_transit_t7_station_icons_updated/directional_vehicle_icons/generic_vehicle/vehicle_train.svg`.
- Generic fallback icons also exist for metro and tram in `athens_transit_t7_station_icons_updated/directional_vehicle_icons/generic_vehicle/`.

## T7 Piraeus Loop Note

- The T7 station list includes the six inbound loop stops before `Akti Poseidonos`.
- Use the coordinates embedded in `athens_fixed_rail_station_coordinates.md` as the current working reference for those six stops.
